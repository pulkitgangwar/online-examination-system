## Requirements

- Postgresql 13 installed
- Pgadmin
- Nodejs v16.13.2
- yarn v1.22.18

## Development Mode

- add a .env file and add database url
- start postgresql service
- run command (yarn dev)
- server with nodemon will start on port 3000


## Production Mode

- add a .env file and add database url
- start postgresql service
- run command (yarn start)
- server will start on port 3000
